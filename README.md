# OneFinder

OneFinder is a donation and request platform that connects donors with people in need.

## Features
- Login and Signup
- Donate items
- Receive requests
- Categories: Food, Clothes, Books, Devices
- Search option
- Dashboard
- Gemini AI support

## Tech Stack
- Flutter
- Firebase Authentication
- Cloud Firestore
- Firebase Hosting
